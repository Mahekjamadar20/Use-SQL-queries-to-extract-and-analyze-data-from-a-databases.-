#  SQL for Data Analysis — E-commerce Dataset

##  Objective
Use SQL queries to explore and analyze structured E-commerce data to uncover customer and sales insights.

## Tools
- MySQL / PostgreSQL / SQLite
- DBeaver / MySQL Workbench

## Key Operations
- SELECT, WHERE, GROUP BY, HAVING
- JOINS (INNER, LEFT)
- Subqueries & Aggregate functions (SUM, AVG)
- Views for reusable insights
- Index creation for optimization

## Outcome
Enhanced practical understanding of SQL data analysis — from query design to optimization and reporting.
